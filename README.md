# 📌 Accredian Clone (Accordion Website)

## 🚀 Setup Instructions

Follow these steps to run the project locally:

1. Clone the repository:

```bash
git clone https://github.com/Jagruti-dhole18/Accredian-Clone
```

2. Navigate to the project folder:

```bash
cd accredian-clone
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```

---

## 💡 Approach Taken

* Built using **Next.js** for fast performance and server-side rendering.
* Created reusable components for the accordion UI.
* Used React state management (`useState`) to handle open/close functionality.
* Followed clean and modular folder structure for scalability.
* Styled using modern CSS/Tailwind for responsive design.
